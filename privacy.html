Privacy Policy

Last updated: April 2026

JSON Cleaner Pro does not collect, transmit, or share any personal data. Everything stays on your device.

What we store locally

The Chrome Extension stores the following data on your device using chrome.storage.local:

  • Your preferred theme (dark or light)
  • Your preferred indent setting (2 spaces, 4 spaces, or tab)
  • A history of up to 10 recent JSON inputs, stored on your device only

This data never leaves your browser. It is not transmitted to any server, third party, or analytics service.

The web app (app.html) stores nothing — no cookies, no local storage, no session data.

How JSON is processed

All JSON cleaning happens entirely within your browser using JavaScript. When you paste JSON into the tool:

  • It is processed locally by code running in your browser tab
  • It is never sent to any external server
  • It is never logged, stored remotely, or analysed

This means the tool is safe to use with sensitive API responses, database exports, and internal data.

External requests

The only external request made by the tool is loading fonts from fonts.googleapis.com. This is a standard web font request that sends your browser's User-Agent string to Google. No JSON content or personal data is included in this request.

If you need zero external requests (e.g. on an air-gapped network), the tool functions correctly without internet — fonts fall back to system defaults.

Permissions (Chrome Extension)

The extension requests the following permissions:

  • clipboardRead / clipboardWrite — used only when you click the Paste or Copy buttons
  • storage — used to save your theme preference and history locally on your device

No other permissions are requested. The extension has no access to the content of web pages you visit.

Third parties

We do not use any analytics, tracking, advertising, or crash reporting services. No third-party code runs in the extension or web app beyond Google Fonts.

Changes to this policy

If this policy changes, the updated date at the top of this page will reflect it. We will not change the policy in ways that result in data collection without prominently noting it.
