Privacy Policy

Last updated: April 2026

✔ JSON Cleaner Pro processes all data entirely within your browser. No JSON data is ever sent to any server.

What We Collect

JSON Cleaner Pro does not collect, transmit, or store any personal data or JSON content you process. All cleaning operations run locally in your browser using JavaScript.

Chrome Extension Storage

The Chrome Extension stores the following data locally on your device using chrome.storage.local:

  • Your last 10 JSON cleaning operations (stored locally, never transmitted)
  • Your theme preference (dark/light)
  • Your indentation preference (2 spaces, 4 spaces, tab)

This data never leaves your device and can be cleared at any time using the "Clear all" button in the History drawer.

Web Tool (GitHub Pages)

The web-based tool at this site processes all JSON in your browser. No data is sent to any server. The page is served as a static file from GitHub Pages.

Third-Party Services

This site loads Google Fonts (DM Sans, DM Mono, Syne) from fonts.googleapis.com. This is a standard CDN request and is subject to Google's Privacy Policy. No other third-party services are used.

Analytics

This site does not use any analytics, tracking pixels, cookies, or third-party advertising scripts.

Children's Privacy

JSON Cleaner Pro is a developer tool intended for professional use. We do not knowingly collect data from children under 13.

Changes to This Policy

If we make material changes to this policy, we will update the date at the top. Continued use of the tool after changes constitutes acceptance.

Contact

Questions about this privacy policy? Open an issue on the GitHub repository.
